# DriveLytix Project

Welcome to the DriveLytix monorepo. This project is divided into three main components:

## Project Structure

- **[mobile/](./mobile)**: The main React Native (Expo) application for automotive diagnostics and telemetry.
- **[backend/](./backend)**: Node.js (Express) API server for data synchronization and cloud features.
- **[web/](./web)**: React (Vite) web dashboard for visualizing telemetry data on the web.

## Getting Started

### Mobile App

```bash
cd mobile
npm install
npm run android # or ios / start
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Web Dashboard

```bash
cd web
npm install
npm run dev
```
